Html이란 무엇인가?

Hypertext Markup Language (HTML) is the standrd Markup language for creating web pages and web applications.

How to Do a Line Break in HTML. To do a line break in HTML, use the br tag. Simply place the tag wherever you want to force a line break.

Since an HTML line break is an empty element, there's no closing tag.
